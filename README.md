# cordova-plugin-minify-code
  

# Install

```
$ cordova plugin add cordova-plugin-minify-code
```
  
# Usage
  
1. Install this plugin in the Cordova project.  
2. The plugin will minify the source code at build time.  
  
# Platforms

```
ios, android, windows
```
